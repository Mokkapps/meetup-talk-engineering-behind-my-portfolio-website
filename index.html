Motivation for Version 3

  • Show my creativity and make a website that is a true expression of myself
  • Design as much as possible myself without using pre-designed templates
  • Make the site as fast and accessible as possible
  • Provide a foundation to make the website easily extendable and adjustable
  • Make the website fully responsive
  • Integrate blog in website

Tech Stack Version 3

GatsbyJS: Static Site Generator

Why GatsbyJS?

  • I love using React and it has a rich ecosystem of libraries, components, etc.
  • It uses GraphQL which I wanted to gain some more practical experience with
  • Very good documentation, plugins & starters
  • Can be easily combined with several APIs, CMS and more

Continuous Integration

Travis CI

Deploy & test my website each time I push to my git master branch

Configuration example (travis.yml):

      
                  script:
                    - npm run lint
                    - npm run test:e2e:ci
                    - npm run build

                  deploy:
                    provider: script
                    script: "curl -X POST -d '' https://api.netlify.com/build_hooks/5ba3c8da1f12b70cbbcaa1a3"
                    skip_cleanup: true
                    on:
                      branch: master
